# DECEPTIVE

A 2D physics puzzle game where the player suffers through obstacles, dodges it memorizes the timing of it and ultimately loses it's temper.

# Description
## It's a replica of LEVIL DEVIL game made by uncept. This game has 50 levels where each levels comes with different mechanics and challenges. It is a very fun game to play!

## Game Mechanics
- Random obstacles popping and pushing
- Spikes which kills the player
- Moving Platforms
- Pillars that make rage baits
- Warpe that can teleport the player
- Trampoline
- Sawe

## Goal
- Make the player touch the portal that takes you to the next level

## Control
- W for Jump
- A and D for horizontal movement
- Warning: It can be different according to the levels!

## Built With
- Unity 2d
- C#
- ShaderLab
- Piskel
